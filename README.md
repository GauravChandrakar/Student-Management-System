# Student-Management-System

## Overview
A web-based Student Management System developed using ASP.NET Core MVC, C#, Entity Framework Core, and MySQL.

## Technologies
- ASP.NET Core MVC
- C#
- Entity Framework Core
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Features
- Student Registration
- View Students
- Update Student
- Delete Student
- Search Student
- Responsive User Interface

🚧 Currently under development.
